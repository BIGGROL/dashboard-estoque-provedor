# Dashboard de Estoque por Prioridade

Projeto em Python para análise de estoque mínimo e máximo de produtos.

## Tecnologias

- Python
- Pandas
- Matplotlib

## O que o projeto faz

- Lê um relatório CSV exportado do IXC.
- Classifica produtos conforme a situação do estoque.
- Gera gráficos automáticos.
- Exporta um relatório tratado.

## Arquivos gerados

- produtos_por_prioridade.png
- estoque_atual_minimo_maximo.png
- reposicao_ate_maximo.png
- estoque_min_max_tratado.csv
